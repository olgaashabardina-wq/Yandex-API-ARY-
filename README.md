# Yandex API ART — генерация изображений

## Зачем этот проект

Проект показывает, как с помощью Python и YandexART API можно быстро собрать рабочий прототип сервиса генерации изображений по текстовому запросу.

Подходит для таких задач:
- генерация иллюстраций под конкретный сценарий;
- создание визуалов для контента, публикаций и презентаций;
- тестирование AI-генерации в прикладных проектах;
- подготовка основы для Telegram-бота или внутреннего инструмента;
- изучение интеграции с YandexART API на практике.
Ниже показан собственный промпт и результат (жёлтый электросамокат под дождём).  
---

## Состав репозитория

- `art_quick.py` — основной Python-скрипт для генерации изображения через YandexART API.

## Требования

Для запуска проекта потребуется:
- Python 3.10+
- аккаунт в Yandex Cloud
- `folder_id`
- IAM-токен

---

## Быстрый старт (локально)

1. Установить зависимости:

python -m pip install requests

$env:FOLDER_ID = "ВАШ_FOLDER_ID"
$env:IAM_TOKEN = "ВАШ_ПОЛНЫЙ_IAM_TOKEN"

python art_quick.py

Результат будет сохранён в result.png.

Используемые технологии

YandexART (Foundation Models)

(опционально) Telegram BotFather + Telegram Bot API

(опционально) RuSender для рассылок

- проект является демо-примером и может быть расширен до бота, веб-приложения или массовой генерации

## Результат

В проекте реализован рабочий пример генерации изображений через YandexART API с использованием Python. Настроен базовый сценарий: отправка промпта, обращение к API и сохранение результата. Репозиторий можно использовать как основу для дальнейшего развития — например, для Telegram-бота, веб-сервиса или AI-инструмента для генерации изображений.
